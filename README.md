# Project Scientific Visualization

## Project Overview
This project explores **students' impulse buying behavior on TikTok Shop** using scientific visualization techniques. The goal is to identify key factors influencing impulse buying and present them interactively using Streamlit.

## Dataset
The dataset consists of survey responses from students about their experience and behavior on TikTok Shop. It has been cleaned and preprocessed to calculate factor scores and overall impulse buying score.

## Structure
- `app.py`: Main page of the Streamlit dashboard.
- `pages/member-impulse-factors.py`: Personal page showing your visualizations, interpretations, and sub-objective.
- `data/responses.xlsx`: Cleaned survey data.
- `requirements.txt`: Python libraries needed.

## How to Run
1. Clone the repository.
2. Install requirements: `pip install -r requirements.txt`
3. Run Streamlit app: `streamlit run app.py`
