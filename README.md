# TikTok Shop Impulse Buying Behavior Dashboard

This project presents an interactive Streamlit application developed to analyze
students' impulse buying behavior on TikTok Shop.

## Objective
To identify the key factors influencing students' impulse buying behavior
through scientific visualization techniques.

## Features
- Interactive filters for demographics
- Five different scientific visualizations
- Clear interpretation of impulse buying factors
- Streamlit-based interactive dashboard

## Tools Used
- Python
- Streamlit
- Pandas
- Matplotlib
- Seaborn

## Dataset
Survey responses collected from university students using Google Forms.

## Deployment
The application is deployed using Streamlit Cloud.
